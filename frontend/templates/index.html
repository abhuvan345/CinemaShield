How It Works

1

Upload

Producer uploads a movie file through the secure dashboard.

2

Shard & Encrypt

Video is split into segments, each encrypted with AES and hashed with SHA-256.

3

Distribute Key

A unique Fernet key is generated. Producer shares it with the authorised theatre.

4

Secure Playback

Theatre decrypts shards just-in-time in memory. No plaintext file ever touches disk.
